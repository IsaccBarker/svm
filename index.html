Simple (language) Virtual Machine


SVM aims to be a embeddable JVM. It doesn't do much, but you can integrate it for some functionality. The goal is to eventually support Java SE 17.

This isn't a project to have a community (although I'm not against that), but rather to further my own knowledge. I would be honored if you use it, but nonetheless I feel you should know my motivations.


A project by Milo Banks.
