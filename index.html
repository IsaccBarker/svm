Simple (language) Virtual Machine

SVM aims to be a embeddable JVM. It doesn't do much, but you can integrate it for some functionality. The goal is to eventually support Java SE 17. Steps have been taken so that SVM can run on any platform that supports the C standard library, and it's written with no external dependencies. It builds fast, and is easy to integrate into your application, with almost everything documented.

This isn't a project to have a community (although I'm not against that), but rather to further my own knowledge. I would be honored if you use it, but nonetheless I feel you should know my motivations.

A project by Milo Banks.
